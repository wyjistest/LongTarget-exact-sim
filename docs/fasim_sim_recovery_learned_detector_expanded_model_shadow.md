## Decision

Decision: `collect_better_features`.

If held-out recall only improves by using oracle-only target rows, the candidate-eligible metrics remain the relevant runtime-feasible view. If the learned shadow cannot beat the current guard on that view, keep the hand-written guard and collect better features or more real-corpus contrastive data.
